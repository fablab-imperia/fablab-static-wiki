Porta NOT

Porta logica che inverte il segnale in ingresso.

Questa porta logica ha un solo ingresso ed una uscita che sarà 1 se l'ingresso è 0 o 0 altrimenti.

Segue la tavola di verità:

INPUT OUTPUT A NOT A 0 1 1 0
